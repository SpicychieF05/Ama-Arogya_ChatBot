About the Project

Bridging Healthcare Gaps in Rural Odisha

Ama Arogya is an AI-driven public health chatbot specifically designed for rural communities in Odisha. The project addresses critical healthcare accessibility challenges by providing 24/7 health guidance in local languages through widely available platforms like WhatsApp.

Technical Architecture

  • NLP Engine: Rasa Open Source for intent recognition and dialogue management
  • Backend: FastAPI with SQLAlchemy for robust data handling
  • Database: SQLite for local deployment and PostgreSQL for production
  • Messaging: Twilio WhatsApp API integration
  • Analytics: Real-time dashboard for usage monitoring
